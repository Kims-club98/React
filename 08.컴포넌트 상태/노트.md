### useEffect란? 
  - useEffect는 컴포넌트가 화면에 나타날 때, 업데이터 될 때 또는 사라질 때 특정 동작을 자동으로 실행하게 해주는 리엑트 함수( == 이벤트 발생 시 실행되는 함수)

  - 주로 데이터 불러오기(Rest API 호출 -> == Back-End의 영역: SpringBoot, Express), 타이머, 이벤트 등록/해제, 브라우저 저장소 접근에 사용

### useEffect가 필요한 이유???
  - 리엑트 컴포넌트 ==> 화면에 보이는 부분(UI)만 그려주고
  - 실 네트워크 요청(서버에서 데이터 받아오기)이나 브라우저 작업(스크롤 조정, 저장 등)은 따로 처리해야 한다.
  - 이런 일들을 부수효과(side effect)라고 부르며, 이때 useEffect가 필요하다!

  ```js
  // useEffect(콜백함수,[의존성 배열])

    useEffect(()=>  {
      // >> 여기에 입력(하고싶은 부수효과를 작성하는 란!) ▼▼▼
      // 랜더링이나 count값이 변경 될때마다 실행된다.(이벤트 발생 시 실행)

    },[count]);
    return (
      <>
      <p>현재 카운트: {count}</p>
        <button onClick{() => setCount(count+1)}>증가</button>
        <button onClick{() => setCount(count-1)}>감소</button>
      </>
    )
  ```
    - 리엑트에서 화면 그리기 외에 추가로 실행할 작업을 등록하는 곳
    - 의도하지 않은 효과 vs 부수효과
